About Exam Editor

THIS TOOL HAS NOT BEEN TESTED IN ANY REAL EXAM YET, WE ADVISE UTMOST CAUTION FOR CRITICAL EXAMS.

Exam Editor is a free, open source tool for creating and/or automatically marking multiple choice exams.

You can find the source code on Github at Markus-Lehr/angular-exam-grader.

How do I use it?

You essentially have two options. Either create an exam from scratch (adding questions, insert existing pdf pages), which can then be exported as pdf and printed. Or you can take your existing multiple choice exams and simply create a grading sheet that fits your existing exam's questions.

Either way, your students simply need to cross their answers on the grading sheet, which is the only essential grading material that you need to collect.

After you're done and the exam has been conducted, scan in your grading sheets in greyscale and with 150 DPI as individual images (e.g. JPG), and import them into the grading tool. You may specify how you want to count points or change other modality options before starting the grading process.

For every inserted image, the tool calculates each question's points and totals them.

Where is my data stored?

Exam Editor manages data completely offline in your browser's indexedDB. We do not transmit any data to any server - we don't even have a backend server for this project.

Therefore, we advise you to backup your exams yourself by exporting them as JSON files. Note that any images or PDFs that you inserted in the exam will not be exported that way in order to save space.

Which exams can I create with the tool?

Essentially any multiple choice exam.

The editor supports inline LateX commands (e.g. $\sum_{{'{'}}0 \leq i \leq k{{'}'}} i$ becomes ),

as well as importing whole PDFs. Therefore, should the tool not satisfy your needs, don't use it and import whatever you created elsewhere.
